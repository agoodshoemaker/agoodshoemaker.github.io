# [mathematics;algebra;factorization]

Heuristics to factorize algebraic expression can be broadly classified into the following headings:

- common factor in all terms
	- $ax + abx + a = a(x + bx + 1)$
- grouping terms
	- $1 + a + b + ab = (1 + a) + b(1 + a) = (1 + a)(1 + b)$
- use of identities
	- $a^2 + 2ab + b^2 = (a + b)^2$
- splitting a term
	- $x^2 + 7x + 12 = x^2 + 3x + 4x + 12 = (x + 3)(x + 4)$
- completing the square / adding and subtracting terms
	- $x^4 + x^2 + 1 = (x^4 + 2x^2 + 1) - x^2 = (x^2 + 1 + x)(x^2 + 1 - x)$
	- $x^4 + 4 = (x^4 + 4x^2 + 4) - 4x^2 = (x + 2x + 2)(x - 2x +2)$
- remainder theorem followed by long-division
	- $f(x) = x^3 - 2x^2 + 4x - 3$; $f(1) = 0 \implies (x - 1)$ is a factor
- remainder theorem applied multiple times
	- $f(x) = x^3 -6x^2 + 11x - 6$; $f(1) = f(2) = f(3) = 0 \implies (x - 1)(x - 2)(x - 3)$ is a factor; as the degree of $f(x)$ is 3, this is the only factor
- symmetry in expressions (also called reciprocal expressions)
	- $x^4 + 2x^3 + 2x^2 + 2x + 1 = x^2\Bigg[x^2 + \cfrac{1}{x^2} + 2\Bigg(x + \cfrac{1}{x}\Bigg) + 2\Bigg] = x^2\Bigg[\Bigg(x + \cfrac{1}{x}\Bigg)^2 + 2\Bigg(x + \cfrac{1}{x}\Bigg)\Bigg]$
	 $= x^2\Bigg(x + \cfrac{1}{x}\Bigg)\Bigg(x + \cfrac{1}{x} + 2\Bigg) = (x^2 + 1)(x + 1)^2$
- cyclic and homogeneous expressions
	- $a(b^2 - c^2) + b(c^2 - a^2) + c(a^2 - b^2) = k(a - b)(b - c)(c - a)$
	- setting $a = b$, we observe that the expression becomes zero, implying that $a - b$ is a factor
	- as the expression is cyclic, $b - c$ and $c - a$ are also factors
	- this implies that $(a - b)(b - c)(c - a)$ is a factor; the degree of this factor is 3
	- as the expression is also homogeneous with degree 3, we don't have any factor other than the one given above
	- $k$ can be found out by setting suitably small values for $a, b, c$ such as $0, 1, -1$
- coefficients of the polynomial and the nature of the root
	- polynomial with integer coefficients: rational roots have to satisfy divisibility tests
	- polynomial with rational coefficients: irrational roots come in conjugate pairs
	- polynomial with real coefficients: complex roots come in conjugate pairs

This list is not exhaustive. We are implicitly using the Fundamental theorem of Algebra, espcially while dealing with the remainder theorem.
