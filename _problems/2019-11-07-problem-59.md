**(Eotvos competition-1894)**


Let $x$ and $y$ be integers. Prove that one of the expressions 

$$
2x + 3y \ \ \text{ and } \ \ 9x + 5y
$$ 

is divisible by $17$ if and only if so is the other.

<!--more-->

<hr><br>

Observe the following:

$$
17(x + y) = 4(2x + 3y) + (9x + 5y)
$$

$17$ will divide $2x + 3y$ if and only if $17$ divides $9x + 5y$.
